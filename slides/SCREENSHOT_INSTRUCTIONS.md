## 🚀 QUICK STEPS:

### 1. Open Each HTML File in Browser

```bash
# Navigate to slides folder
cd /Users/tarang/CascadeProjects/windsurf-project/RevuIQ/slides

# Open in browser (Mac)
open slide1.html
open slide2.html
open slide3.html
```

Or just double-click each file to open in your browser.

### 2. Take Screenshots

**On Mac:**
- Press `Cmd + Shift + 4`
- Press `Spacebar` (to capture full window)
- Click on the browser window
- Screenshot saved to Desktop

**On Windows:**
- Press `Windows + Shift + S`
- Select area
- Save screenshot

### 3. Insert into Google Slides

1. Go to your Google Slides presentation
2. Click "Insert" → "Image" → "Upload from computer"
3. Select the screenshot
4. Resize to fit slide
